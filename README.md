# Superstore_Sales_Dashboard
Interactive Power BI dashboard analyzing Superstore sales performance, profit trends, customer segments, and regional insights.

## Tools Used

* Power BI
* Superstore Dataset

## Dashboard Features

* Sales Trend Over Time
* Top Products by Sales
* Sales Contribution by Category
* Regional Sales Performance
* KPI Metrics
* Interactive Slicers

## Key Insights

* The Technology category generated the highest sales among all categories.
* The West region contributed the highest revenue.
* Sales showed an overall upward trend over time.
* A small number of top-performing products contributed significantly to total sales.

## Recommendations

* Focus marketing efforts on high-performing categories to maximize revenue.
* Improve sales strategies in lower-performing regions.
* Promote best-selling products through targeted campaigns.
* Monitor product performance regularly to identify growth opportunities.

## Outcome

This project demonstrates the use of Power BI for data visualization and storytelling by transforming sales data into actionable business insights and recommendations.
